Ordlista för programmering

  1. Algoritm – steg-för-steg-instruktion för att lösa ett problem.
  2. Array – ordnad samling av element.
  3. API – gränssnitt för att använda en tjänst eller bibliotek.
  4. Argument – värde som skickas in i en funktion.
  5. Asynkron – sker oberoende av huvudflödet.
  6. Bibliotek – samling kod som kan återanvändas.
  7. Bug – fel i programmet.
  8. Cache – tillfällig lagring för snabb åtkomst.
  9. Class – mall för objekt i objektorienterad programmering.
  10. Compile – översätta kod till maskin­talang.
  11. Conditional – sats som utförs endast om ett villkor är uppfyllt.
  12. CSS – språk för styling av HTML.
  13. Data­struktur – sätt att organisera data (t.ex. list, stack).
  14. Debugga – hitta och åtgärda fel.
  15. Deklarera – definiera en variabel eller funktion.
  16. Dependency – något koden är beroende av.
  17. DOM – dokumentets objektmodell i webbläsare.
  18. Environment – körmiljö för program.
  19. Exception – undantag vid körning.
  20. Funktion – återanvändbar kodbit som utför en uppgift.
  21. Framework – struktur som hjälper utveckling.
  22. Git – versionshanteringssystem.
  23. HTML – språk för webbsidors struktur.
  24. IDE – integrerad utvecklingsmiljö.
  25. Immutable – oföränderlig.
  26. Iteration – upprepad körning (loop).
  27. JSON – textformat för datautbyte.
  28. Lambda – anonym funktion.
  29. Library – se bibliotek.
  30. Loop – upprepad kodkörning.
  31. Module – kod­enhet som kan importeras.
  32. Namespace – område för identifierare för att undvika konflikt.
  33. Object – instans av en klass.
  34. Operator – symbol som utför en operation (t.ex. +, ==).
  35. Override – skriva om en metod i subklass.
  36. Package – samling moduler/bibliotek.
  37. Parameter – variabel i funktionsdefinition.
  38. Pointer – variabel som pekar på minnesadress (i C/C++).
  39. Polymorphism – objekt kan anta olika former.
  40. Recursion – funktion som anropar sig själv.
  41. Repository – lagringsplats för kod (t.ex. på GitHub).
  42. Runtime – tiden då program körs.
  43. Script – kod som tolkas i realtid.
  44. Semantics – betydelse av kod.
  45. Stack – LIFO-data­s truktur.
  46. Syntax – regler för kodens form.
  47. Variable – namn för ett värde som kan ändras.
  48. Version – specifik utgåva av programvara.
  49. Web API – API som tillhandahålls via webben.
  50. XML – strukturerat dataformat.
